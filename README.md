# 🍴 Food Delivery Analytics
This project analyzes restaurant performance, customer behavior, delivery efficiency, and payment trends across multiple cities.   It aims to provide actionable insights for improving customer satisfaction and optimizing delivery operations.

## 📘 Overview
This project analyzes restaurant performance, customer behavior, delivery efficiency, and payment trends across multiple cities.  
It aims to provide actionable insights for improving customer satisfaction and optimizing delivery operations.

---

## 🧩 Dataset Description
The dataset captures food delivery transactions with the following columns:

| Column | Description |
|--------|--------------|
| Restaurant | Name of the restaurant |
| FoodItem | Ordered food item |
| Quantity | Number of items ordered |
| OrderValue | Value before charges |
| TotalAmount | Final billed amount |
| DeliveryTime_Minutes | Delivery duration |
| CustomerRating | Rating given by customer |
| City | Location of the order |
| PaymentMethod | Mode of payment |
| OrderDate | Date of the order |

---

## 🎯 Objectives
- Identify top-performing restaurants by revenue and ratings  
- Analyze customer preferences and payment trends  
- Evaluate delivery efficiency and its impact on satisfaction  
- Discover relationships between order value, delivery time, and ratings  

---

## 🧠 Methodology
1. **Data Cleaning:** Handle missing values, standardize categories, remove duplicates  
2. **Exploratory Data Analysis (EDA):**  
   - Univariate: Delivery time, ratings, payment methods  
   - Bivariate: Delivery time vs. rating  
   - Multivariate: City, restaurant, payment method vs. order value  
3. **Visualization:**  
   - Bar charts, histograms, heatmaps, pie charts  
4. **Dashboard Creation:**  
   - Interactive Power BI dashboard with slicers for city, restaurant, and payment method  

---

## 📊 Key Metrics
- Total Revenue & Average Order Value  
- Average Delivery Time & Late Delivery %  
- Customer Rating Distribution  
- Top 10 Restaurants by Revenue  

---

## 🚀 Tools Used
- **Python:** Pandas, NumPy, Matplotlib, Seaborn  
- **Excel:** Pivot Tables, Charts  
- **Power BI:** Dashboard creation and interactivity  

---

## 💡 Insights & Outcomes
- Faster deliveries lead to higher ratings  
- Wallet and UPI payments dominate urban orders  
- Top restaurants contribute 60% of total revenue  
- Data-driven recommendations for operational improvement
